This repository contains official code examples for the book  
**PIC Microcontroller Essentials** by Ferdinand Godfrey.

## About the Book
This book focuses on practical, register-level programming of
PIC16 enhanced mid-range microcontrollers using MPLAB X.

Topics include:
- GPIO and digital I/O
- Timers and interrupts
- ADC and PWM
- UART communication
- Interrupt handling
- MPLAB X simulator-based debugging

## Repository Structure
Each folder corresponds to a topic in the book.
All projects are provided as MPLAB X `.X` projects.

## Tools Required
- MPLAB X IDE
- XC8 Compiler
- PIC16 enhanced mid-range devices
